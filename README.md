# 📄 Resume Analyzer

A machine learning-based application that analyzes resumes and matches them with relevant job roles by extracting key skills and information.

---

##  Features

-  Resume parsing (PDF/DOCX support)
-  Skill extraction using NLP techniques
-  Job role matching based on skills
-  Keyword analysis for better screening
-  Simple and user-friendly interface

---

##  Tech Stack

- Python
- Natural Language Processing (NLP)
- Machine Learning
- Libraries: NLTK / spaCy / scikit-learn
- Streamlit / Flask (if you used it)

---

│── requirements.txt # Dependencies

---

## ⚙️ How It Works

1. Upload a resume (PDF/DOCX)
2. Extract text from the resume
3. Process text using NLP techniques
4. Identify key skills and keywords
5. Match with relevant job roles
